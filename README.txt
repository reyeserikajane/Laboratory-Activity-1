
=========================
INSTRUCTIONS FOR ACTIVITIES
=========================

PREREQUISITES (For All Activities)
----------------------------------
1. Python Installed:
   Make sure you have Python 3 installed on your system.
   You can verify it by running:
       python --version
   or
       python3 --version

2. Text Editor or IDE:
   Use any text editor or IDE (e.g., VS Code, Sublime Text, PyCharm, Notepad) to create and edit the .py files.

=====================================
ACTIVITY 2: Client-Server Interaction
=====================================
1. Save the server code as: server.py
2. Save the client code as: client.py

3. Open TWO terminal windows:
   - In Terminal 1, run the server:
         python server.py
   - In Terminal 2, run the client:
         python client.py

4. Observe the message exchange in both terminals.
   This demonstrates the client-server relationship (ClientApplication communicatesWith ServerService) described in MEL.
